## 4. 주요 위험요인
### food_poisoning [S1]
- 위험 이유: 식중독·식품위생 사고 위험(medium)이 입력 조건과 연결됩니다.
- 통제: 영업허가·신고·등록 대상 여부와 푸드트럭/임시영업 조건을 확인한다.
- 통제: 냉장·보온, 손위생, 조리구역 오염방지, 알레르기·표시 기준을 점검한다.
- 통제: 식중독 의심 발생 시 보고·보존·조사 연락체계를 운영본부에 둔다.
### temporary_structure_collapse [S8] [S10] [S11] [S5] [S6]
- 위험 이유: 임시구조물·부스·무대·트러스 전도/붕괴 위험(high)이 입력 조건과 연결됩니다.
- 통제: 독립부스, 복층, 리깅, 대형 구조물은 베뉴 승인과 구조검토 대상 여부를 확인한다.
- 통제: 설치·해체 작업 전 작업신고서와 고소작업/중량물 작업 계획을 확인한다.
- 통제: 야외 구조물은 풍속 기준과 중지·철거 트리거를 둔다.
### temporary_electrical_fire_shock [S1] [S10] [S6]
- 위험 이유: 임시전기 감전·누전·화재 위험(high)이 입력 조건과 연결됩니다.
- 통제: 분전반, 누전차단기, 케이블 포설, 통로 횡단 보호, 접지 상태를 점검한다.
- 통제: 전기 작업은 지정협력업체·자격 보유자·베뉴 기술지원 절차를 따른다.
- 통제: 물 사용 구역, 야외 우천, 푸드부스 주변 전기설비를 별도 점검한다.
### crowd_density_high [S3] [S2] [S1]
- 위험 이유: 군중 밀집도 상승 위험(high)이 입력 조건과 연결됩니다.
- 통제: 구역별 수용능력과 순간 최대 인원 기준을 사전에 정한다.
- 통제: 입장·퇴장·대기열을 분리하고 병목 구간에는 일방통행 또는 우회 동선을 둔다.
- 통제: 밀집 구역에는 스태프를 고정 배치하고 혼잡 단계별 통제 문구를 준비한다.
### ingress_egress_bottleneck [S3] [S1]
- 위험 이유: 입퇴장 병목 위험(high)이 입력 조건과 연결됩니다.
- 통제: 게이트별 처리량, 대기열 길이, 우회 동선을 사전 산정한다.
- 통제: 입장 피크와 프로그램 종료 피크의 스태프 배치를 분리한다.
- 통제: 임시 펜스·바리케이드는 피난 방향을 막지 않도록 배치한다.

## 5. 적용하지 않은 항목
- 공연법 계열: 공연 프로그램이 없다고 입력되어 공연 관련 기준을 적용 후보에서 제외했습니다.
- LPG·가스 안전 기준: LPG·가스를 사용하지 않는 조건이므로 관련 기준을 제외했습니다.
- 경비업법 계열: VIP 경호·보안검색·민간경비 조건이 없어 관련 기준을 제외했습니다.

## 4. 주요 위험요인
### fire_hazard_hot_work_lpg [S1] [S8] [S12] [S6]

- 통제: 위험물·화기·가스 반입 허가 대상 여부를 사전에 확인한다.
- 통제: 소화기 위치, 화기 이격거리, 가스용기 전도방지, 누출점검을 확인한다.
- 통제: 베뉴별 금지 물질과 반입 절차를 참가업체 매뉴얼에 반영한다.
### food_poisoning [S1]
- 위험 이유: 식중독·식품위생 사고 위험(medium)이 입력 조건과 연결됩니다.
- 통제: 영업허가·신고·등록 대상 여부와 푸드트럭/임시영업 조건을 확인한다.

- 통제: 식중독 의심 발생 시 보고·보존·조사 연락체계를 운영본부에 둔다.
### temporary_structure_collapse [S8] [S10] [S11] [S5] [S6]
- 위험 이유: 임시구조물·부스·무대·트러스 전도/붕괴 위험(high)이 입력 조건과 연결됩니다.
- 통제: 독립부스, 복층, 리깅, 대형 구조물은 베뉴 승인과 구조검토 대상 여부를 확인한다.
- 통제: 설치·해체 작업 전 작업신고서와 고소작업/중량물 작업 계획을 확인한다.
- 통제: 야외 구조물은 풍속 기준과 중지·철거 트리거를 둔다.
### temporary_electrical_fire_shock [S1] [S10] [S6]
- 위험 이유: 임시전기 감전·누전·화재 위험(high)이 입력 조건과 연결됩니다.
- 통제: 분전반, 누전차단기, 케이블 포설, 통로 횡단 보호, 접지 상태를 점검한다.
- 통제: 전기 작업은 지정협력업체·자격 보유자·베뉴 기술지원 절차를 따른다.
- 통제: 물 사용 구역, 야외 우천, 푸드부스 주변 전기설비를 별도 점검한다.
### crowd_density_high [S3] [S2] [S1]
- 위험 이유: 군중 밀집도 상승 위험(high)이 입력 조건과 연결됩니다.
- 통제: 구역별 수용능력과 순간 최대 인원 기준을 사전에 정한다.
- 통제: 입장·퇴장·대기열을 분리하고 병목 구간에는 일방통행 또는 우회 동선을 둔다.
- 통제: 밀집 구역에는 스태프를 고정 배치하고 혼잡 단계별 통제 문구를 준비한다.

- 통제: CCTV는 설치 목적, 촬영 범위·시간, 관리책임자 연락처, 수탁자 명칭·연락처 안내를 확인한다.
- 통제: QR/출입증/등록대행 수탁자와 재위탁 제한, 접근권한, 접속기록, 암호화, 현장 단말 잠금 기준을 문서화한다.

- 공연법 계열: 공연 프로그램이 없다고 입력되어 공연 관련 기준을 적용 후보에서 제외했습니다.
- 식품위생법 계열: 식음료 판매·제공이 없다고 입력되어 식품 관련 기준을 제외했습니다.
- LPG·가스 안전 기준: LPG·가스를 사용하지 않는 조건이므로 관련 기준을 제외했습니다.
- 경비업법 계열: VIP 경호·보안검색·민간경비 조건이 없어 관련 기준을 제외했습니다.

- 이 결과는 법률 자문이나 허가 판단을 대신하지 않습니다.
